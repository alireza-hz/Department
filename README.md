# Department Management API

A clean architecture ASP.NET Core 8 Web API for managing internal letters, users, notes, and attachments.  
It supports JWT authentication, role-based access, and a hybrid file storage system (Redis + File System).

### Features
- JWT Authentication (Register/Login)
- Letter and Workflow Management
- Notes and Letter Types
- File Attachments stored in Redis or on Disk
- Repository Pattern with Clean Architecture

### Tech Stack
.NET 8 • EF Core 8 • SQL Server • Redis • Swagger • CORS & Policy
